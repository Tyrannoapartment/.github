# Tyranno Apartment Corp.

> **사람을 위한 친근한 서비스를 만듭니다.**

<br>

티라노아파트는 사용자 중심의 서비스 경험을 만드는 소프트웨어 개발 회사입니다.
우리는 복잡한 문제를 단순하고 직관적인 제품으로 해결하는 것을 목표로 하며, 뛰어난 UI/UX와 안정적인 기술을 기반으로 다양한 디지털 서비스를 설계하고 개발합니다.

특히 Flutter와 Dart를 중심으로 프론트엔드와 백엔드를 모두 개발하는 풀스택 개발 역량을 갖추고 있으며, 빠른 제품 개발과 높은 유지보수성을 동시에 추구합니다.

기술은 단순히 기능을 제공하는 것을 넘어, 사람들의 일상에 자연스럽게 스며들어 긍정적인 경험을 만들어야 한다고 믿습니다. 우리는 이러한 철학을 바탕으로 사용자가 편하게 사용할 수 있는 서비스와 제품을 만들어가고 있습니다.

## 🚀 What We Do

- **서비스 제품 개발:** 사용자 중심의 모바일 및 웹 서비스를 기획하고 설계하며 실제 제품으로 구현합니다.
- **Flutter 기반 크로스 플랫폼 개발:** 하나의 코드베이스로 모바일, 웹, 데스크톱 환경에서 동작하는 제품을 개발합니다.
- **Fullstack Development with Dart:** Flutter와 Dart 생태계를 기반으로 프론트엔드와 백엔드를 통합적으로 설계하고 개발합니다.

## 🛠️ Technology & Design

저희는 최고의 사용자 경험을 제공하기 위해 다음과 같은 기술과 도구를 적극적으로 활용합니다.

- **UI/UX Design:** `Figma`, `Adobe Creative Suite`
- **Frontend:** `Flutter`, `Dart`
- **Backend:** `Dart`, `Shelf`
- **Infrastructure:** `Google Cloud Platform`, `Firebase`
- **CI/CD:** `GitHub Actions`
